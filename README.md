# Enviromentor Agents — Stackd AI

AI agent stack for Enviromentor — gutter cleaning and garden maintenance, Sydney.

## Agents
- **CEO Agent** — orchestrates all other agents (every 5 min)
- **Lead Agent** — qualifies new enquiries
- **Outreach Agent** — 10 personalised emails per weekday morning
- **Follow-up Agent** — 4-step nurture over 21 days
- **Scheduling Agent** — appointment reminders, rescheduling
- **Review Agent** — Google review requests post-job
- **Report Agent** — daily 5pm summary, weekly Monday summary
- **Social Agent** — weekly Facebook + LinkedIn posts
- **Briefing Agent** — daily morning/evening brief to Wander
- **Health Agent** — hourly system check, email alerts

## Deployment
- Platform: Railway (new project for Enviromentor)
- Schedule: every 5 minutes
- Timezone: Australia/Sydney
